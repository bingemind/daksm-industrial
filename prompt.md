### PROJECT ROLE: Lead Frontend Developer
### PROJECT: DASKM Engineering Industries $15k Vanilla Transformation

Build a premium, high-performance industrial website using Pure HTML5, Tailwind CSS (via PostCSS/CLI), and Vanilla JavaScript.

#### 1. EXECUTION STRATEGY (STRICT)
- **Incremental Edits:** Work section-by-section. Do not overwrite whole files.
- **Edit Tool Only:** Use the Edit tool to apply diffs to existing code.
- **Bespoke Quality:** Focus on clean, documented code that a non-dev can understand.

#### 2. DECOUPLED ARCHITECTURE
- **config.js:** Create a file to hold the `PRODUCT_DATA` array and `CALCULATOR_CONSTANTS`.
- **Logic Separation:** The UI should be generated by a `renderProducts()` function in `app.js` that loops through the config.
- **CSS Variables:** All colors must be defined as variables in `:root` and `.dark` classes in the CSS.

#### 3. DUAL-THEME SPECS (Theme Toggle)
- **Light Mode ("Precision"):** White/Grey bg, Tech Blue (#0056B3) accents. Reference `images/valve-precision-light.webp`.
- **Dark Mode ("Powerhouse"):** Charcoal/Navy bg, Safety Orange (#FF6B00) accents. Reference `images/valve-powerhouse-dark.webp`.

#### 4. THE "HOOK": VALVE CALCULATOR
- Build a standalone `Calculator` component in `js/calculator.js`.
- Inputs: Type (Ball, Gate, Globe), NPS Size (0.5" - 24"), Class (150-2500).
- Logic: Functional programming style. It should return Weight and Torque values based on the config.

#### 5. FILE STRUCTURE
- `/index.html`
- `/css/styles.css` (Tailwind + Custom Vars)
- `/js/config.js`
- `/js/app.js`
- `/js/calculator.js`
- `/public/images/` (Referencing: valve-precision-light.webp, valve-powerhouse-dark.webp, valve-exploded-view.webp, valve-iso-cert-bg.webp)
